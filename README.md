Project Overview

This project focuses on analyzing student mental health and academic factors using SQL Server and Tableau.
The analysis helps understand how different factors like academic pressure, study satisfaction, sleep duration, study hours, and financial stress affect students based on student count distribution.

The final result is an interactive Tableau dashboard published on Tableau Cloud.
